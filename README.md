# Credit_card_fraud_detection_dashboard
End-to-end fraud detection using python,sql and power bi
This project focuses on detecting and analyzing fraudulent credit card transactions using Python, SQL, and Power BI.
The goal is to identify fraud patterns, analyze transaction behavior, and present insights through an interactive dashboard.
- Python (Pandas, NumPy)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)
- Jupyter Notebook
- fraud_detection_project/
│── data/                  # Raw dataset
│── notebooks/             # EDA & analysis notebooks
│── scripts/               # Data cleaning & feature engineering
│── outputs/               # Cleaned dataset
│── dashboard/             # Power BI file (.pbix)
│── README.md
1. Data Cleaning using Python
2. Feature Engineering (Hour, Day extraction)
3. SQL Analysis to identify fraud patterns
4. Power BI Dashboard for visualization
5. - Fraud transactions are extremely rare (~0.17%)
- Peak fraud activity occurs around specific hours
- Lower transaction values show higher fraud frequency
- Strong class imbalance observed in dataset
- Power BI Dashboard includes:
- Fraud Percentage KPI
- Fraud by Hour Analysis
- Fraud vs Normal Distribution
- Transaction Amount Analysis
- Interactive Filters
- Python scripts (scripts/)
- Jupyter notebooks (notebooks/)
